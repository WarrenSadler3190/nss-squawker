# Squawker

## Summary

This is the basis of the pair projects for capstone 3, for cohort 3 at Nashville Software School.

This is a basic twitter clone.

## Project Setup

1. Create the databases:
  `rake db:create:all`
2. Run the migrations:
  `rake db:migrate`
3. Start the app:
  `rails s`
